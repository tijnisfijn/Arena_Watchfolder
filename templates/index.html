Resolume Arena — Watch Folder Sync

How to use Arena Watchfolder

This app syncs local folders of media files to layers in Resolume Arena. Drop videos and images into a folder, and they appear as clips in Arena.

Getting started

  1. Connect — Make sure Arena's webserver is enabled (Preferences > Webserver), then click Connect. The default host/port works if Arena is on the same machine.
  2. Add a mapping — Click + Add Folder Mapping, pick a folder with your media files and choose which Arena layer to sync them to.
  3. Sync — Click Sync Now to load the folder's files into Arena. Only new/removed files are touched — existing clips keep all their effects and settings.

Sets & profiles

A set is a named profile (e.g. "Thursday Night", "Friday Night") with its own folder-to-layer mappings. When you switch sets, the current clip settings are saved automatically, and the new set's settings are restored. Great for different shows or events.

Watch mode

Click Start Watch on a mapping to continuously monitor the folder. When you add or remove files, the layer updates automatically. The orange border and pulsing dot indicate active watching.

Collect from Arena

Use Collect to copy an Arena layer's clip files into your local folder, creating a local backup of your show. You can also use Collect All at the top to collect every layer at once into organized subfolders.

  • Collect (per mapping) — Copies clip files from the mapped Arena layer into the mapping's folder. Generated sources (Checkered, Lines, etc.) have no files to copy but are captured in the snapshot.
  • Collect All — Scans all Arena layers and copies clips into <dest>/<composition>/<deck>/<layer>/ subfolders, creating a mapping for each layer.

Generated sources

Arena's built-in content generators (Solid Color, Checkered, Lines, Abstract Field, Gradient, etc.) are fully supported. When you collect or save settings, generated sources are captured with all their parameters — colors, density, grid size, rotation, etc.

When you Force Sync to a different layer, generated sources are automatically re-created in their original slots and their settings are restored. This means you can reorganise generators across layers just like regular video clips.

Save & Restore settings

  • Save All Settings — Takes a snapshot of all clip settings across every mapping at once: effects and their parameters, transform, speed, direction, cue points, blend modes, and more. Also captures generated source parameters. Saving all layers together keeps everything in sync.
  • Restore Settings — Applies the last saved snapshot back to the clips on a specific layer. Clips are matched by filename, generated sources by slot position. If a clip returns after being removed, its settings come back too.

Settings are also saved/restored automatically when switching sets.

What gets saved

Effects (Blur, Color Balance, etc.) with all parameter values, Transform (position, scale, rotation), speed & direction, cue points (in/out), play mode (autopilot), beat loop, opacity, resize mode, audio (volume, pan), clip name, trigger style, transition, dashboard knobs, source parameters (colors, density, grid size for generated sources), and more.

What is NOT saved

  • Automation — Arena UI feature, not available through the API
  • Cue markers — the diamond markers in the timeline are UI-only
  • Live playback position — the playhead moves every frame (cue in/out points are saved)
  • Layer-level settings — layer effects, blend mode, opacity, and transitions are not captured (snapshots are per-clip only)
  • Composition settings — output resolution, FPS, etc. are not affected

In short: everything you set on a clip is saved — including generated sources with their parameters. Everything set on the layer or composition is not.

Composition & Deck lock

Use these options to prevent accidental syncs to the wrong show:

  • Lock composition — Syncs are blocked if Arena has a different .avc file loaded.
  • Lock deck — Syncs are blocked if a different deck is active. If the locked deck no longer exists (e.g. you opened a new composition), the lock auto-clears.

Shutting down

Always quit the app when you're done. If you leave it running in the background (especially in watch mode), it will keep syncing files to Arena. This can cause problems if you open a different composition, because the app may still be pushing clips to the old layer mappings.

Use the Quit Server button (top right), the Quit button (desktop app), or press Ctrl+C in the terminal.

Tips

  • Files sync alphabetically into clip slots — rename files with number prefixes (01_, 02_, ...) to control the order.
  • Use the ✎ edit button on a mapping to change its folder path or layer number.
  • The Sync All button (top right of the mappings card) syncs every mapping in the current set at once.
  • When clips return to a layer after re-syncing, you'll be asked whether to restore their previous settings or keep them fresh.
  • Generated sources survive the full Collect → Force Sync → Restore round-trip, including custom colors, grid sizes, and effects.
  • Always use local folders for your media — never point mappings directly at a network share. If you collaborate remotely, use a synced folder (Dropbox, Google Drive) so files land on your local drive first.
